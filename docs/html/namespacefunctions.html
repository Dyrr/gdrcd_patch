functions\file (   $file)

Genera il path per il set di funzioni richiesto.

Parametri
[in]$file(string) nome del set di funzioni da caricare
Restituisce
(type) il path completo della risorsa

La funzione genera il path completo di una risorsa in modo darenderne più facile l'inclusione. La funzione nasce come funzione "interna" da reciclare all'interno di funzioni più specifiche per il tipo di risorse richieste.

Esempio
$file = \functions\file(
'core/db/mysqli'
);
//Ritornerà come valore di $file: root/system/inc/functions/core/db/mysqli.inc.php
Si veda anche
functions::file
