Foglio di stile con le variabili dei css.

Nel file sono impostate tutte le variabili utilizzate dal preprocessore di css, sia come variabili vere e proprie a livello di valori sia, come ovvero dei blocchi di impostazioni comuni che verranno poi richiamate ed incluse all'intenro di alcuni stili.
Sarà per cui possibile prendere ed impostare un colore come variabile per esempio:

1 @set {
2  default_padding: 10px;
3 }

Per impostare la variabile da richiamare poi con:

1 div.container:$(default_padding);

Oppure impostare un set di proprietà:

1 @mixin font_base
2 {
3  font-size:14px;
4  font-family:Whitney, Helvetica Neue, Helvetica, Arial, sans-serif;
5  line-height:20px;
6 }

Richiamate poi così:

1 div.pagina_presenti_estesa
2 {
3  ul.elenco_presenti
4  {
5  li.presente
6  {
7  @include font_base;
8  }
9  }
10 
11 }
