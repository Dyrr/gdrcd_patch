data (   $attr,
  $value,
  $regex = true 
)

Stampa a video un attributo data di un tag HTML.

Parametri
[in]$attr(string) Il nome dell'attributo data da creare
[in]$data(string|int) Il valore dell'attributo
[in]$regex(bool) Opszione per il controllo del valore dell'attributo tramite regex

La funzione analizza il valore dell'attributo data indicato dell'invio dei dati alla funzione e in base al valore decide se stampare a video l'attributo, rendendo più intuitivo la stampa a video degli attributi di quel tipo nei template.
La funzione di default ha impostato una regex di controllo del valore dell'attributo che accetta come caratteri validi solo lettere numeri e i caratteri - _ #.
La funzione ha una regola specifica per l'attibuto delay nel caso questo abbia valore 0 impostandone il valore a none.

Esempio
<?php
$v = array(
'function' => 'main',
'target' => '',
'delay' => 0
);
?>
<a href="test.php>"
<?php data('function',$v['funzione']); ?>
<?php data('target',$v['target']); ?>
<?php data('delay',$v['delay']); ?>
>test</a>
//genererà come output
<a href="test.php" data-function="main" data-delay="none">test</a>

landURL (   $modulo,
  $azione = null,
  $parametri = null 
)

Helper per ls generazione dei link.

Parametri
[in]$modulo(string) il modulo da eseguire usando '/' come separatore per le cartelle
[in]$azione(string) l'operazione da far eseguire al modulo
[in]$parametri(array/string) array o stringa con i parametri per la querystring
Restituisce
(string) l'url nel formato richiesto

Il metodo genera l'url nel formato utilizzato del CMS (cartella/modulo//azione/querystring) tramite i valori passati alla funzione.
Il metodo in particolare permette di passare i parametri per la querystring in diversi formati per essere Il più flessibile possibile per le varie esigenze. I parametri possono essere passari come:

  • array nel formato array('nome_parametro_1' => 'valore','nome_parametro_2' => 'valore');
  • stringa nel formato nome_parametro_1=valore,nome_parametro_2=valore;
  • stringa con la querystring tradizioneale ?nome_parametro_1=valore&nome_parametro_2=valore. Il metodo individuerà automaticamente la modalità con cui è passata la lista dei parametri e la formatterà per la generazione dell'url completo.
Si veda anche
system/lib/helpers/_alias.php per ulteriori informazioni sugli helpers

out (   $data)

Stampa a video una variabile filtrata in uscita.

Parametri
[in]$data(mixed) la variabile in entrata da filtrare

La funzione stampa a video una variabile filtrandola nel caso si rilevi che questa sia una stringa

Esempio
<?php
$data = '<script tyle="javascript">alert(\'XSS\');</script>';
out($data);
?>
